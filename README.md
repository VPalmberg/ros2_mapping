# ROS2 Mapping Project

## 🇬🇧 English

This project focuses on creating a custom simulation environment and performing autonomous mapping using ROS 2 (Humble).

The main tasks:
- Create a custom environment in Gazebo
- Use TurtleBot3 for simulation
- Implement an autonomous mapping node
- Generate a map using SLAM (Cartographer)
- Save the final map (map.pgm + map.yaml)

The project demonstrates how a robot can explore and map an unknown environment using LiDAR data.

## 🇷🇺 Русский

Этот проект посвящён созданию пользовательской среды симуляции и автономному построению карты с использованием ROS 2 (Humble).

Основные задачи:
- Создать собственную среду в Gazebo
- Использовать TurtleBot3 для симуляции
- Реализовать автономный mapping node
- Построить карту с помощью SLAM (Cartographer)
- Сохранить карту (map.pgm + map.yaml)

Проект демонстрирует, как робот может исследовать и картировать неизвестную среду с помощью LiDAR.
